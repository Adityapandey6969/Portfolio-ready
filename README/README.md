# Project Documentation

This folder contains project-level documentation for the portfolio application.

It is intentionally focused on the codebase, setup, structure, assets, and deployment workflow rather than any personal profile content.

## Files

- `PROJECT_OVERVIEW.md`: high-level explanation of the app, stack, routes, and folder structure
- `SETUP_AND_DEPLOYMENT.md`: local development, production build, and deployment steps

## Recommended Reading Order

1. `PROJECT_OVERVIEW.md`
2. `SETUP_AND_DEPLOYMENT.md`
